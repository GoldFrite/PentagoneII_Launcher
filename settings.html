Configuration pour le Jeu

Configuration minimale requise

Système d'exploitation : Windows 7 ; OS X El Capitan (10.11) ; Ubuntu 14.04 ; Fedora 24 ; Debian 8
Architecture : 32 bits (x64, x86, ARM)
Mémoire vive totale : 4 Go
Processeur : Intel Core i3-3210 ; AMD A8-7600 APU ; équivalent
Carte graphique intégrée au processeur* : Intel HD Graphics 4000 (Ivy Bridge) ; AMD Radeon R5 (Kaveri) + OpenGL 4.4
Carte graphique dédiée* : NVIDIA GeForce 400 Series ; AMD Radeon HD7000 + OpenGL 4.4
Stockage alloué : 700 Mo HDD

Configuration minimale recommandée

Système d'exploitation : Windows 10 ; macOS Big Sur (11.x ou 10.16) ; Ubuntu 20.04 ; Fedora 33 ; Debian 10
Architecture : 64 bits (x64, x86, ARM)
Mémoire vive totale : 8 Go
Processeur : Intel Core i5-4690 ; AMD A10-7800 APU ; équivalent
Carte graphique intégrée au processeur* : Non recommandé
Carte graphique dédiée* : NVIDIA GeForce 700 Series ; AMD Radeon RX 200 (hors puces intégrées) + OpenGL 4.5
Stockage alloué : 1 Go SSD

* : Un processeur peut avoir une carte graphique intégrée. Celles-ci sont souvent moins performantes qu'unr carte graphique dédiée, mais sont tout de même acceptables. De plus, il n'est pas necessaire d'avoir une carte graphique intégrée au processeur pour jouer sur le serveur.
